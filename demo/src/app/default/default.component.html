Welcome

Here we are, the Angular 2 version of the Angular UI Bootstrap library. This library is being built from scratch in Typescript using the Bootstrap 4 CSS framework.

To get started quickly you can follow installation instructions or just browse demos for the existing components.

As with Bootstrap 4 and Angular 2, this library is a work in progress. Please check out our roadmap and list of issues to see all the things we are implementing. Feel free to make comments there.
